# Accessibility Guide (접근성 가이드)

> 시니어 사용자를 위한 접근성은 이 프로젝트의 최우선 요구사항입니다.

## WCAG AA 준수 (최소)

### 글씨 크기
- **본문 최소**: 18px (1.125rem)
- **채팅 메시지**: 20px (1.25rem)
- **버튼 텍스트**: 18px (1.125rem)
- **헤더**: 24px+ (1.5rem+)
- `font-size`를 `rem` 단위로 사용하여 브라우저 확대 지원

### 색상 대비
- **일반 텍스트**: 최소 4.5:1 대비 비율
- **큰 텍스트 (18px+)**: 최소 3:1 대비 비율
- **UI 요소/아이콘**: 최소 3:1 대비 비율
- 색상만으로 정보를 전달하지 않음 (아이콘 + 텍스트 병행)

### 터치 타겟
- **최소 크기**: 48x48px (WCAG 2.2 Target Size)
- **권장 크기**: 56x56px+ (시니어 사용자 고려)
- 터치 타겟 간 최소 8px 간격

### 키보드/보조기술
- 모든 인터랙티브 요소 키보드 접근 가능
- 포커스 인디케이터 명확히 표시 (최소 2px outline)
- aria-label, aria-live 적절히 사용
- 스크린 리더 호환성 확보

## 시니어 특화 가이드

### 인터랙션
- 더블 탭 대신 싱글 탭
- 스와이프 제스처 최소화
- 명확한 피드백 (로딩 상태, 성공/실패)
- 실행 취소 기능 제공

### 콘텐츠
- 짧고 명확한 문장
- 전문 용어 대신 쉬운 말
- 아이콘에는 항상 텍스트 레이블 병행
- 에러 메시지는 해결 방법을 함께 제시
