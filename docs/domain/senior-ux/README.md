# Senior UX Guide (시니어 UX 가이드)

## 핵심 원칙

### 1. 단순함이 최고다
- 화면당 하나의 주요 액션
- 불필요한 옵션 제거
- 명확한 시각적 계층구조

### 2. 실수해도 괜찮다
- 모든 액션에 실행 취소 제공
- 확인 다이얼로그는 명확하게
- 에러 메시지는 친절하고 구체적으로

### 3. 기다림을 알려준다
- 로딩 상태를 명확히 표시
- AI 응답 생성 중 타이핑 인디케이터
- 진행률 표시 (가능한 경우)

### 4. 읽기 쉬워야 한다
- 최소 18px 글씨 크기
- 높은 대비 비율 (4.5:1 이상)
- 줄 간격 1.5 이상
- 최대 줄 너비 65자

### 5. 따뜻하고 친근하다
- 존댓말 사용
- 격식체보다 친근한 존댓말
- AI 캐릭터에 이름과 성격 부여
- 긍정적이고 격려하는 톤

## 레이아웃 가이드

### 모바일 (주요 타겟)
```
┌─────────────────────┐
│  [로고] 시니어챗      │  ← 헤더 (간결)
├─────────────────────┤
│                     │
│  AI: 안녕하세요!     │  ← 메시지 영역 (넓게)
│  어떤 도움이         │
│  필요하세요?         │
│                     │
│  나: 오늘 날씨       │
│  알려줘              │
│                     │
├─────────────────────┤
│  [메시지 입력...]    │  ← 입력 영역
│  [마이크] [전송]     │  ← 큰 버튼
└─────────────────────┘
```

### 색상 팔레트

| 용도 | 색상 | 이유 |
|------|------|------|
| AI 메시지 배경 | 따뜻한 베이지 (#FFF8F0) | 친근하고 편안한 느낌 |
| 사용자 메시지 배경 | 부드러운 파랑 (#E3F2FD) | 구분이 명확하면서 편안 |
| 주요 버튼 | 깊은 파랑 (#1565C0) | 높은 대비, 신뢰감 |
| 텍스트 | 진한 회색 (#212121) | 높은 가독성 |
| 배경 | 밝은 크림 (#FFFEF7) | 눈의 피로 감소 |

## 접근성 체크리스트

- [ ] 모든 텍스트 18px 이상
- [ ] 색상 대비 4.5:1 이상
- [ ] 터치 타겟 48px 이상
- [ ] 키보드로 모든 기능 접근 가능
- [ ] 스크린 리더 호환
- [ ] 확대 200%에서도 레이아웃 유지
- [ ] 애니메이션 최소화 (prefers-reduced-motion 존중)
